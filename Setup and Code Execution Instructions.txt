#Οδηγίες Ρύθμισης και Εκτέλεσης Κώδικα

##Ρύθμιση Υπολογιστή
1.Για να ρυθμίσω το Bash terminal στα Windows, ακολούθησα τα εξής βήματα:
α.Ενεργοποίηση Υποσυστήματος Windows για Linux (WSL):
β.Πίνακα Ελέγχου > Προγράμματα > Ενεργοποίηση ή Απενεργοποίηση Δυνατοτήτων των Windows.
γ.Ενεργοποιήσα την επιλογή “Υποσύστημα Windows για Linux” στη λίστα.
δ.Έκανα επανεκκίνηση τον υπολογιστή και έπειτα εγκατάστασα το Ubuntu 16.04
2. Για να ρυθμίσω το τερματικό:
α. Ανοίγω το εγκατεστημένο τερματικό του Ubuntu.
β. Εκτελώ τις παρακάτω εντολές στο τερματικό:
sudo apt-get update
sudo apt-get install libmpich-dev

##Οδηγίες Εκτέλεσης Κώδικα 
Χρησιμοποιώ τον επεξεργαστή κειμένου Sublime Text και το τερματικό του Ubuntu για τη μεταγλώττιση και την εκτέλεση του κώδικα.
#Μεταγλώττιση
1.Για να μεταγλωττίσω τον κώδικα για την τοπολογία δακτυλίου, χρησιμοποιώ την παρακάτω εντολή:
mpicc matrix_matrix_ring.c -o mmr -DN=3 -lm
mpicc: Ο wrapper compiler MPI για προγράμματα C. Περιλαμβάνει τις απαραίτητες βιβλιοθήκες MPI κατά τη διαδικασία μεταγλώττισης.
matrix_matrix_ring.c: Το αρχείο πηγαίου κώδικα για το πρόγραμμα που υλοποιεί τον πολλαπλασιασμό μητρώων χρησιμοποιώντας τοπολογία δακτυλίου.
-o mmr: Καθορίζει το όνομα του εκτελέσιμου αρχείου. Σε αυτή την περίπτωση, mmr είναι το όνομα του εκτελέσιμου αρχείου που δημιουργείται από τον μεταγλωττιστή.
-DN=3: Ορίζει έναν προεπεξεργαστή N με την τιμή 3. Είναι ισοδύναμο με την προσθήκη #define N 3 στην κορυφή του αρχείου πηγαίου κώδικα. Το N αντιπροσωπεύει το μέγεθος των μητρώων που θα χρησιμοποιηθούν στον πολλαπλασιασμό μητρώων.
-lm: Συνδέει τη βιβλιοθήκη μαθηματικών (libm), είναι απαραίτητο καθώς το πρόγραμμα χρησιμοποιεί μαθηματικές συναρτήσεις.
2. Για να μεταγλωττίσω τον κώδικα για την τοπολογία πλέγματος, χρησιμοποιώ την παρακάτω εντολή:
mpicc matrix_matrix_grid.c -o mmg -DN=3 -lm
mpicc: Ο wrapper compiler MPI για προγράμματα C.
matrix_matrix_grid.c: Το αρχείο πηγαίου κώδικα για το πρόγραμμα που υλοποιεί τον πολλαπλασιασμό μητρώων χρησιμοποιώντας τοπολογία πλέγματος.
-o mmg: Καθορίζει ότι το εκτελέσιμο αρχείο θα πρέπει να ονομάζεται mmg.
-DN=3: Ορίζει έναν μακροεντολέα προεπεξεργαστή N με την τιμή 3, καθορίζοντας το μέγεθος των μητρώων για τον πολλαπλασιασμό μητρώων.
-lm: Συνδέει τη βιβλιοθήκη μαθηματικών.

##Εντολές Εκτέλεσης
1.Για να εκτελέσω τον κώδικα για την τοπολογία δακτυλίου, χρησιμοποιώ την παρακάτω εντολή:
mpirun -np 4 ./mmr < input.txt
mpirun: Η εντολή που χρησιμοποιείται για την εκτέλεση προγραμμάτων MPI. Ξεκινά πολλαπλές διεργασίες του καθορισμένου προγράμματος MPI.
-np 4: Καθορίζει τον αριθμό των διεργασιών που θα ξεκινήσουν. Σε αυτή την περίπτωση, θα δημιουργηθούν 4 διεργασίες.
./mmr: Δείχνει το εκτελέσιμο αρχείο που θα εκτελεστεί, το οποίο σε αυτή την περίπτωση είναι το mmr αρχείο που δημιουργείται από τη μεταγλώττιση του matrix_matrix_ring.c.
< input.txt: Ανακατευθύνει την τυπική είσοδο για να διαβάσει από ένα αρχείο με όνομα input.txt. Αυτό το αρχείο περιέχει τα δεδομένα εισόδου που απαιτεί το πρόγραμμα, όπως τα στοιχεία των μητρώων για τον πολλαπλασιασμό.
2.Για να εκτελέσω τον κώδικα για την τοπολογία πλέγματος, χρησιμοποιώ την παρακάτω εντολή:
mpirun -np 4 ./mmg < input.txt
